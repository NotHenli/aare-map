# Bilder für die Popups

Ein Bild erscheint automatisch im Popup eines Streckenpunkts, sobald hier eine
JPG-Datei mit der passenden ID liegt (Querformat, ca. 800×500 px empfohlen):

| Datei | Punkt |
|---|---|
| `vermietung-schwaebis.jpg` | Aarebootsvermietung (Logo/Foto) |
| `schwaebis.jpg` | Einstieg Schwäbis |
| `vermietung-uttigen.jpg` | Aareschlauchboot (Logo/Foto) |
| `uttigenwelle.jpg` | Uttigenwelle |
| `uttigen.jpg` | Einstieg Uttigen |
| `auguetbruecke.jpg` | Auguetbrücke |
| `eichholz.jpg` | Ausstieg Eichholz |
| `marzili.jpg` | Ausstieg Marzili |
| `schwelle.jpg` | Wehr Schwellenmätteli |

Fehlt eine Datei, wird das Popup einfach ohne Bild angezeigt.
